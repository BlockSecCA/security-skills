# security-skills

Dev security skills for Claude Code: code analysis, vulnerability detection, and pentest methodologies.

## Install

```bash
claude plugin marketplace add BlockSecCA/security-skills
claude plugin install security-skills@security-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| joern-analysis | Code security analysis with Joern CPG: import code, navigate relationships, trace data flows, find vulnerabilities |

## Requirements

- [Joern](https://joern.io) installed and running as a server (`joern --server`)
- [joern-mcp](https://github.com/BlockSecCA/joern-mcp) MCP server registered
